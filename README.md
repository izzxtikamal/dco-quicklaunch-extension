# dco-quicklaunch-extension
DCO QuickLaunch Extension - Quick access to essential DCO tools

Quick Launch is a browser extension designed specifically for DCO to streamline daily workflows and eliminate friction in accessing critical tools and resources. By transforming the browser's new tab page into a centralized command center, Quick Launch provides instant, one-click access to over 25 essential DCO tools, significantly reducing the time spent navigating to frequently-used internal resources.
